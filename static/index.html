How Auto Download Works

The app checks dependencies on launch and can auto-download missing tools into your user profile runtime folder (no admin required).

Troubleshooting Ideas

  1. If auto-install fails, click Auto Install Missing again after confirming internet access.
  2. If corporate/firewall rules block downloads, manually install FFmpeg and yt-dlp using the links above.
  3. If security software quarantines binaries, allow the runtime folder and retry.
  4. After manual installs, restart the app so dependency checks refresh cleanly.
